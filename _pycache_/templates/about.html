About Movie Magic

Movie Magic is a smart movie ticket booking system that allows users to browse, select, and book movie tickets online. Our platform provides a seamless booking experience with real-time seat selection.

Built with Flask and deployed on AWS cloud services including EC2, DynamoDB, and SNS for notifications.

Features:

  • Browse latest movies
  • Real-time seat selection
  • Instant booking confirmation
  • Secure payment processing
  • Booking history
